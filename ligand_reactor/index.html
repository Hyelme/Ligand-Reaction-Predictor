Ligand Reaction Predictor(LRP) is

A website that helps the study of the ligand by drewing the predicted response results right away when new ligand emerges. The prediction information provided by this website is not a solid result of 100 percent, so please use only for reference when you use it for your research.
If this is your first time using this site, Click on the tab that looks like top and use it step by step, otherwise scroll down and use them.

Step1. Select File

Step1 is the file selection step.
Click the button labeled Select File. And Browse the file containing the SMILES value and select that.
On this occasion, selected files can only be in .txt or .smi format. And the SMILES value entered in the file is recognized as one SMILES per line.

Step2. Select a type of Learning Machine

Step2 is the learning machine selection stage. CNN and DMLP are types of deep learning algorithms, each learning the ligand data in different ways.
Select the prediction type in the desired learning machine, then click the 'PREDICTION' button. It takes about 20 seconds to 1 minute for the results to be outputed, depending on the number of choices of the prediction type.

On this occasion, prediction type can be double-selected but the learning machine type cannot be double-selected. For example, if you select the prediction type for the CNN learning machine, you cannot select the prediction type for the DMLP learning machine.
When you does click this AUC button, you can do check the accuracy of the learning rate.

The graph below shows the accuracy for each learning machine. Performance of CNN model is 74%, 79% and 71% repectively, and performance of DMLP model is 50%, 50% and 50% repectively. The CNN model is relatively accurate more than the DMLP model. (Hover over the graph to zoom in.)
